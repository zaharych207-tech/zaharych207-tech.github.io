1. Демонстрация z-index и clamp()

z-index управляет наложением позиционированных элементов. Чем больше значение, тем выше слой.
clamp(min, предпочтительное, max) — функция, которая ограничивает значение между минимумом и максимумом. Здесь используется для размера шрифта.
z-index: 1 (красный)
z-index: 3 (зеленый) — выше всех
z-index: 2 (синий)
Этот текст использует font-size: clamp(1rem, 4vw, 2rem). Попробуйте изменить ширину окна — размер будет плавно меняться от 1rem до 2rem в зависимости от 4vw.

2. Свойства position на четырёх блоках

Каждый блок имеет padding, margin, border, border-radius, box-shadow, overflow (из раздела 7.1).
• static — значение по умолчанию, смещения не работают.
• relative — смещается относительно своего обычного места.
• absolute — вырывается из потока, позиционируется относительно ближайшего позиционированного предка (здесь — контейнер .position-demo).
• fixed — прикреплён к окну браузера (правый нижний угол).
static (по умолчанию)
top/left не действуют. Обычный блок в потоке.
relative
Сдвинут на 15px вниз и 10px вправо от своего исходного места.
absolute
Позиционируется относительно родительского контейнера (.position-demo). top:40px, right:10px.
fixed
Прикреплён к окну: bottom:20px, right:20px. Не двигается при прокрутке.

4. Grid Layout — контейнер и элементы

Grid-контейнер: display: grid.
Свойства контейнера: grid-template-columns: repeat(3, 1fr); (три равные колонки), grid-template-rows: auto auto;, gap: 15px;.
Свойства элементов: grid-column: span 2; (широкий элемент), grid-row: span 2; (высокий элемент). Также показана явная сетка.

Overflow

Свойство overflow: auto создаёт полосы прокрутки при необходимости. В блоке справа текста больше, чем его высота.
